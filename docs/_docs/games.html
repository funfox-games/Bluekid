Games

Games are the core mechanic of Bluekid, it is what allows you to connect students (or friends) to a game to either study or have fun. There are multiple ways to play a kit called gamemodes. Gamemodes are explained later down.

Hosting

Students join off a code. Either from a link, the code directly, or your profile.

Warning

If the code is obtained by someone, they can join from anywhere. However, there are plently of security features implemented such as kicking, hiding the code, and more. If you have any questions, contact me.
