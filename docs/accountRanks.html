Account Ranks and their upgrades

Ever heard of the feature called... account ranks? Account ranks are used for security purposes and give you access more and more as you use Bluekid.
Now the reason you are here is probley because you are asking this: "What does this feature do and why is it important?" Well, below, you will find all the perks to having these ranks!

All features!

Newbie

  • Get access to account
  • Create kits
  • Restricted Game hosting (teachers: Full game hosting)
  • Full game joinning

Intermediate

  • Everything from Newbie
  • Full game hosting
  • Unlock other special features

Advanced

  • Everything from Intermediate
  • Get a special badge (once verified by owner)
  • Get access to WIP features
